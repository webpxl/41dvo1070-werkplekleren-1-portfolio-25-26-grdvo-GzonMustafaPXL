# Portfolio website - Gzon Mustafa (1 DVOD)

Deze repository bevat een **portfolio website** gebouwd met **enkel HTML, CSS en plain JavaScript** (geen CMS, geen frameworks).

## Live publicatie
Publiceer via **GitHub Pages** of **Netlify**. De docent moet de website kunnen openen met 1 URL.

## Structuur
- `/index.html` : hoofdsite
- `/portfolio/` : apart deel (via `https://<jouw-gebruikersnaam>.github.io/<repo>/portfolio/`)
- `/css/styles.css`
- `/js/main.js`
- `/privacy.txt`
- `/assets/` : eigen assets (SVG placeholders + eigen projectbeelden)

## Externe bronnen
- Geen externe libraries.
- Afbeeldingen: SVG placeholders + projectbeelden toegevoegd door de student (eigen werk).

## AI-tools
- Gebruikt: ChatGPT (tekst & basisstructuur van de website).
